Tijdlijn

ARPA
Datum 1969
Het allereerste begin van het internet. Het internet (of arpanet zoals het toen heette) was bedoeld voor het delen van informatie tussen verschillende computers

World Wide Web / HTTP
Datum 1991
Het wereld wijde web (www) maakte het mogelijk om verschillende computers met elkaar te verbinden. Dit ging niet alleen over een paar computers maar over de hele wereld. Hoewel het wereld wijde web oorspronkelijk bedoeld was voor het linken van bestanden groeide het uit naar iets veel groters.

HTML (tags)
Datum 1997
HTML is de taal van het wereld wijde web. In deze taal worden websites geschreven. Voor de komst van CSS (zie CSS) werd deze taal ook gebruikt voor de opmaak van een website. Dit terwijl dit helemaal niet de bedoeling was van HTML.

CSS
Datum 1994
CSS was de oplossing voor het vormgeven van een webpagina. CSS zorgde niet alleen voor een makkelijkere manier van vormgeving maar ook van een loskoppeling tussen website en vormgeving. Dit maakte aanpassingen maken veel efficienter.

Fixed width sites
Datum 1996 - 2004
De eerste personal computers kwamen met een vaste scherm breedte. In 1996 lag dit op 640px, in 2000 werd dit 800px en in 2004 was dit al opgelopen naar 960px. Deze vaste breedtes zorgeden ervoor dat websites gemaakt werden met een vaste breedte.

Mobile web
Datum 2006
Ook voor de opkomst van de smartphones(zie smartphones) waren mobiele telefoons in staat om het internet te bezoeken. Voor deze en de latere smartphones werd het .mobi domein in het leven geroepen. Dit domein hield de 'mobiele' websites apart van de 'gewone' websites. Deze websites konden daardoor beter gebouwd worden met een klein scherm in gedachten.

Smartphones (iPhone)
Datum 2007
Met de aankondiging van de iPhone werd een nieuw tijdperk ingeluid. Dat van de smartphone. Op deze telefoons kon niet alleen gebeld en gesmst worden, maar ook foto's genomen, muziek geluisterd en, misschien wel het belangrijkst, het internet bezocht worden.

Responsive design
Datum 2009
In 2009 heeft Ethan Marcotte de term Responsive web design gecoined. Deze term houd in dat websites zich aanpassen aan het scherm waarin ze getoond worden. Dit kunnen mobiele (dus kleine) of desktopschermen (of te wel grote schermen) zijn; maar ook een broswer scherm dat niet het hele scherm vult op een desktop.
